# WHY

Computers make humans remember where information is stored.

Humans remember ideas, conversations, and experiences—not filenames.

This project exists to bridge that gap by making computers understand intent instead of keywords.

Every feature should answer one question:

Does this reduce friction between a human and their own knowledge?

# NOT NOW
"It would be cool if it also...", DONT IMPLEMENT IT

---

# SEARCH VS AI

AI should not replace search. AI should enhance search.

The system should use the simplest, fastest method capable of answering the user's question.

A filename lookup should take milliseconds.

A complex knowledge question should use the LLM.

The user should never pay the cost of AI when traditional methods are sufficient.

See Decision #002.

---

# OPERABILITY

If this app cannot be used daily on the primary Windows PC (16GB RAM, RTX 5060 Ti), the feature is wrong — even if it demos well on a more powerful machine.

Prefer settings and degraded modes over hard requirements so weaker and stronger machines can still run the same architecture.

See Decision #003.
