# Screenshots

Product UI captures for README / docs (regenerate after major launcher visual changes):

```powershell
npm run build
npx electron scripts/capture-screenshots.js
```

| File | Shows |
|------|--------|
| `launcher-idle.png` | Idle mosaic + empty search |
| `launcher-searching.png` | Query entered (results slot reserved) |

Tray chrome is OS-native; behavior is documented in the README caption.
